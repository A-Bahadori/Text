Title:
Add Customized applicationhost.config File for Optimized IIS Express Project Execution
Description:
This Merge Request includes a customized applicationhost.config file to improve and standardize the project execution process with IIS Express.
Essential Configuration:

Program Arguments Settings:

The path to the customized file must be set in the Program arguments section
Example: /config:D:\Sink\WebApp\applicationhost.confg /site:WebApp


Run/Debug Configuration Settings:

The AppPath variable must be defined in the Environment Variables section
Example: AppPath=D:\Sink\WebApp



These changes ensure that all team members can run the project with consistent and standardized settings, preventing environment-related errors.
Please verify these settings in your development environment before approval.
